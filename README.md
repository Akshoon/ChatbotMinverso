# Minverso - Generador de Contenido LinkedIn

## ¿Qué es?

Una extensión de Chrome que genera posts de alta conversión para LinkedIn B2B usando inteligencia artificial (Gemini).

## ¿Cómo funciona?

1. **Describe tu tema** - Escribe sobre qué quieres crear un post (evento, lanzamiento, caso de éxito, etc.)

2. **Recibe 3 ganchos** - La IA te propone 3 opciones de gancho diferentes:
   - Hook Disruptivo: Desafía normas de la industria
   - Hook Humano: Basado en autoridad de líderes
   - Hook Inmersivo: Enfocado en curiosidad visual/tech

3. **Elige tu gancho favorito** - Responde con "1", "2" o "3"

4. **Obtén el post final** - Recibes:
   - Texto completo listo para copiar
   - Placeholders de etiquetado como `@[CEO de Minverso]` para que agregues los nombres reales
   - Primer comentario estratégico

## Instalación

1. Abre `chrome://extensions/`
2. Activa **"Modo de desarrollador"**
3. Clic en **"Cargar extensión sin empaquetar"**
4. Selecciona esta carpeta

## Configuración

1. Clic en el ícono de configuración (⚙️)
2. Ingresa tu API Key de Gemini ([obtener aquí](https://aistudio.google.com/app/apikey))
3. Guardar
